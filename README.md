# ⚡ Install-manager

Легковесный и быстрый комплекс bash-скриптов для автоматического развертывания и удобного управления сервером Xray (Reality) прямо из терминала.

---

## 🚀 Быстрая установка в одну строку

Запустите эту команду на  сервере (Ubuntu/Debian) с установленным Xray для автоматической интеграции всех утилит:

```bash
bash -c "$(curl -sL https://raw.githubusercontent.com/ffhu8/install-manager/main/install.sh)"
```

 📦 **Что делает инсталлятор:** изолирует исполняемые скрипты в системной директории `/usr/local/share/xray-manager` и создает глобальные символические ссылки для удобного вызова команд из любой точки системы.

---

## 📥 Отдельное скачивание скриптов

Если вам не нужна автоматическая установка и вы хотите скачать скрипты по отдельности в текущую папку, используйте следующие команды:

- **Основной генератор / Инициализатор сервера:**

    ```bash
      curl -sL https://raw.githubusercontent.com/ffhu8/install-manager/main/generate_vless.sh -o generate_vless.sh && chmod +x generate_vless.sh
    ```

- **Скрипт добавления клиентов:**

    ```bash
      curl -sL https://raw.githubusercontent.com/ffhu8/install-manager/main/update_clients.sh -o update_clients.sh && chmod +x update_clients.sh
    ```

- **Скрипт удаления клиентов:**

    ```bash
    curl -sL https://raw.githubusercontent.com/ffhu8/install-manager/main/remove_clients.sh -o remove_clients.sh && chmod +x remove_clients.sh
    ```


## 🛠 Доступные команды управления

После работы полноценного инсталлятора `(install.sh)` вам больше не нужно искать файлы в папках. Просто вводите в консоли:

- **`xray-gen`** - Полная инициализация сервера с нуля. Полностью зачищает старые конфигурации, генерирует новые криптографические ключи Reality, пересоздает технический шаблон и создает первую партию рабочих клиентов.

- **`xray-add`** — Добавление новых пользователей. Быстро генерирует новые конфигурации и ссылки `vless://` без изменения настроек уже существующих клиентов.

- **`xray-rm`** — Безопасное удаление клиентов. Удаляет выбранного пользователя из конфига сервера и стирает файлы его конфигурации с диска.


## 📂 Структура проекта на сервере

- `/usr/local/etc/xray/config.json` — Основной конфигурационный файл службы Xray.

- `/usr/local/etc/xray/clients/` — Папка со сгенерированными текстовыми файлами, содержащими готовые ссылки `vless://` для импорта в клиенты (v2rayNG, Necobox, Streisand и др.).
